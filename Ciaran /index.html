Binary Quest

(Group 21) Programming Project

Binary quest is a logic game. The player must complete their way through a series of levels to complete the game. At the beginning of each level, the player is presented with a ”black box” logic system, consisting of a set of inputs, a set of outputs, and a set of desired states. The user must design a logic system that maps the inputs to the outputs according to the state speciﬁcation.
